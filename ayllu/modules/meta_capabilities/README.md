# Meta-capabilities (CANON)

Mandatory modules; not optional.

## Ministry of State Capacity (Systems & Delivery)

**Purpose:** Make the state execute reliably.

- Delivery Unit (service KPIs, backlog, escalation)
- Evaluation Unit (impact evaluation, A/Bs where appropriate)
- Procurement modernization
- Org design / process engineering
- Civil service training + rotations

**Outputs (mandatory):** Delivery Standard, Program Evaluation Standard, Procurement Playbook.

## Ministry of Future, Risk, and Civilizational R&D

**Purpose:** Treat long-horizon threats/opportunities as first-class.

- National Risk Register + stress testing
- Horizon scanning + scenario planning
- Frontier tech governance (AI, bio, cyber, etc.)
- Experimentation sandboxing (regulatory sandboxes)

**Outputs (mandatory):** Foresight Standard, Critical Risk Preparedness.
