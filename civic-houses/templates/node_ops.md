# How to start a Civic House (node ops) — template

1. Read the starter kit (brand, service catalog, safety, financial controls, dispute resolution).
2. Define your node — name, city, venue model, initial services.
3. Adopt safety_and_conduct and align with venue partner.
4. Set up financial controls (treasury, ledger).
5. Register with the network when registry exists.
6. Launch with at least one concrete service from the catalog.
7. Report metrics per metrics.md.

Ongoing: regular ops review; participate in dispute resolution and protocol updates.
